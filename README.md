# UAct

A Unity Editor Extension for Create and Edit Assets

## v0.1.0 - Aug 2025 Update

In initial release provide tow modules.

## Features

### Batch

- Extract Material
- Assign Texture
- Invert Textures G Channel

### Generator

- Generate ScriptableObject From Excel File.

## How to use

1. Download Unity Package or Souce.
2. Import package or code to your project.
3. Find UAct in menu.
